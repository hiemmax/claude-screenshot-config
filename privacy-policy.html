Screenshot for Claude

Privacy Policy

Last updated: March 2026

No data is collected

Screenshot for Claude does not collect, store, or transmit any personal data. No account is required. No identifying information is ever recorded.

Screenshots are processed in memory only

When you capture a tab, the screenshot exists as a temporary in-memory image for the duration of the transfer — typically a few seconds. It is never written to disk, never stored in the extension's storage, and is discarded as soon as it has been delivered to the Claude.ai input.

Data goes directly to Claude.ai — nowhere else

The screenshot is injected directly into your Claude.ai browser tab using standard browser APIs. It travels through your own browser session, the same way you would paste an image manually. The extension does not proxy, relay, or intercept any of this data.

No analytics, no tracking, no third-party services

The extension contains no analytics libraries, crash reporters, ad networks, or tracking pixels. No third party ever receives information about your usage.

Remote config file

The extension periodically fetches a small JSON file from GitHub Pages to stay up to date with Claude.ai's interface. This file contains only:

  • CSS selectors used to locate Claude.ai's file input
  • A flag to temporarily disable the extension if needed
  • A minimum version requirement
  • The URL to open for new Claude.ai conversations

No user data is included in this request. The request carries no cookies, no identifiers, and no payload — it is a plain GET to a static file. The response is parsed as data only; no code is executed from it.

Permissions used

The extension requests the following Chrome permissions, and only uses them for the stated purpose:

  • activeTab — to capture a screenshot of the tab you are currently viewing
  • scripting — to inject the screenshot into the Claude.ai input field
  • tabs — to identify the active tab and switch to Claude.ai
  • storage — to cache the remote config so it is not fetched on every use
